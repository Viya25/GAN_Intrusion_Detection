# GAN_Intrusion_Detection
GAN-based Intrusion Detection System to handle class imbalance by generating synthetic attack data and improving rare event detection using machine learning.
This project focuses on solving the class imbalance problem in intrusion detection systems using Generative Adversarial Networks (GANs).

In real-world datasets, cyber attacks are rare compared to normal network traffic, which leads to poor performance of machine learning models. To address this, a GAN model is used to generate synthetic attack data that mimics real attack patterns.

The generated data is then combined with the original dataset to train a machine learning model, resulting in improved detection of rare attacks. The performance is evaluated and compared with traditional techniques such as SMOTE.

This project demonstrates how deep learning-based data generation techniques can enhance model performance in imbalanced data scenarios.
