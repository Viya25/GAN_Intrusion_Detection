# GAN-Based Intrusion Detection System

## 📌 Overview
This project uses Generative Adversarial Networks (GANs) to generate synthetic attack data and improve detection of rare cyber attacks.

## 🚀 Features
- Handles imbalanced dataset problem
- Generates synthetic attack data using GAN
- Improves detection performance
- Compares GAN with SMOTE

## 🛠️ Tech Stack
- Python
- Machine Learning (Random Forest)
- Deep Learning (GAN - PyTorch)
- Data Preprocessing & Visualization

## 📊 Dataset
NSL-KDD Intrusion Detection Dataset

## 📈 Results
- Improved recall for rare attack classes
- Better performance compared to baseline model
